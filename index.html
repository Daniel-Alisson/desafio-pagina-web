1. Introdução à Web

  • 1.1 Surgimento da Internet

O surgimento ocorreu durante a Guerra Fria com a ideia de desenvolver uma rede de comunicação para conectar centros de pesquisas e realizar a troca de infromações, resultando no primeiro computador ARPANET. Logo foram desenvolvidas várias mudanças para tornar o sistema mais completo, surgiram protocolos como o TCP/IP e a criação da World Wide Web por TIM Berners-Lee, dessa forma a Internet se populariozou.

Com a criação da World Wide Web (WWW)a internet se tornou acessivel para o publico. E conseguentemente foram desenvolvidos navegadores, Mosaic, Netscape, Internet Explorer, com a intenção de facilitar o acesso a páginas Web. Logo surgiram os primeiros sites, serviços de busca e email e cada vez se tornando um mecanismo de comunicação indispensavel.

Devido ao crescimento acelerado, foi nescessario um padrão e organização da Web, portanto Tim Bernets-Lee desenvolveu o HTML (HyperText Markup Language). O HTML permitiu a navegação fluida com o uso de hiperlinks e outras abordagen. Desde então a linguagem de marcação se tornou a base essencial para o desenvolvimento da Web.
